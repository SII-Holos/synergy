Replace the complete ordered list of pending future Steps in the current Lattice Pathway. Use only while planning or reviewing_pathway. Existing completed, failed, cancelled, current, or executing Steps are preserved and immutable. To retain and revise an existing pending Step, include its id; omit id to create a new Step. This tool changes only the Pathway artifact and pathwayRevision. It does not submit the Pathway, select a Step, bind a Blueprint, or advance workflow state; use lattice_submit for those semantic actions.
