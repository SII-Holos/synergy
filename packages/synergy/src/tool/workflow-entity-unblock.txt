Unblock a blocked workflow entity so it returns to the queue and re-enters the normal event chain. Only available to a Boss session. Use this after investigating the entity's seat session (via session_read / session_search using its seatSessionID) to understand the root cause of the block. If the blocking condition has been resolved (e.g. a seat pool freed, a missing resource is now available, or the block was spurious), call this to restart the entity's pipeline from the initial state.
