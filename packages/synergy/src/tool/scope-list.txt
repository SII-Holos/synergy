List available Synergy scopes (projects + home) so you can choose a valid `scopeID`.

Each entry includes:
- `id`: stable Scope ID; use a project ID as `scopeID` with `session_list` or `session_control create`
- `type`: `home` or `project`
- `name` / `directory`: human-readable project identity
- optional activity stats (`sessionCount`, `latestActivityAt`)

Use this to select a specific project. Workflow:
1. Call `scope_list` (optionally with `query` to filter by name/path/id)
2. Pick the matching project `id`
3. To list its sessions, call `session_list` with `scope: "project"` and `scopeID: "<id>"`
4. To create a session there, call `session_control` with `action: "create"` and `scopeID: "<id>"`

If you already know the absolute project path but not the Scope ID, you may pass `directory` to `session_control create` instead.

Parameters:
- `query`: optional case-insensitive filter over id, name, and directory
- `includeHome`: include the home scope (default true)
- `limit` / `offset`: pagination (max 100 per page)
