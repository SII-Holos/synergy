Create an agenda item — a scheduled task, recurring job, or reminder executed by an agent.

The `prompt` is the instruction the agent follows when the item fires. Write it as if you're briefing another agent on what to do.

Items with triggers (cron, every, at, delay, watch) are automatically activated. Items without triggers start as "pending" — use `agenda_trigger` to run them manually.

By default, when execution completes, the result is delivered back to this session and wakes the agent (wake=true). Set `silent: true` to suppress delivery, or `wake: false` to deliver without waking the agent.

Items are scoped to the current project by default. Set `global: true` to make an item visible from all projects.

For watch triggers, use `kind: "tool"` to poll a Synergy tool (e.g. inspire_jobs) instead of a shell command — this is more reliable and doesn't require external CLI tools.
